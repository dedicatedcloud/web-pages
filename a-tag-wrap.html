Wrap Strings in <a href="" target="_blank"></a>

Paste your URLs (or any strings) in the left box and matching labels in the right box (one per line). Then click Generate to create ready-to-paste anchor tags.

Tip Line 1 ↔ Line 1, Line 2 ↔ Line 2, etc.
One value per line. Empty lines are ignored.
One value per line. If a line is empty, the URL will be used as the label.
Example output:
<a href="https://example.com" target="_blank">Example site</a>
